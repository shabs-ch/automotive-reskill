# Automotive Reskill

AI-powered skills mapping and reskilling tool for German automotive engineers facing displacement.

## Status
Week 1, Day 1 — hello world prototype

## Stack
- Python 3.14
- Streamlit
- Anthropic Claude API
- ChromaDB (Week 2)
- HuggingFace Spaces (Day 3)

## Setup
1. Clone the repo
2. Create .env with ANTHROPIC_API_KEY=your_key
3. pip install -r requirements.txt
4. streamlit run app.py
