# User Stories — v1

## US1 — Transferable skills visibility
As an engineer in transition, I want to see my transferable skills mapped to specific target roles, so that I can confidently present 
my value to recruiters in language they recognise.

## US2 — Role discovery
As an automotive engineer in transition, I want to see which roles — within automotive AI and beyond — best match my background with a clear 
explanation of why, so that I can make the strongest possible next move without abandoning my domain expertise unnecessarily.

## US3 — Personalised learning roadmap
As an engineer overwhelmed by AI course options, I want a personalised, realistic step-by-step learning roadmap that starts from my current skills towards the identified target roles. This shall help me know exactly what to learn next without wasting time on irrelevant content.

## US4 — Bildungsgutschein navigation
As an engineer in Germany, I want to know which courses in my roadmap are eligible for Bildungsgutschein funding and how to apply, so that I can upskill without paying out of pocket.

## US5 — CV reframing
As an automotive engineer applying to new domain or latest roles in automotive, I want my CV bullets rewritten to highlight the relevant experience in the language hiring managers expect, so that my application passes initial screening.

## US6 — Gap understanding
As an engineer actively applying for roles, I want to understand exactly what skills I'm missing for my target role, so that I can close the right gaps instead of applying blindly and getting rejected.
